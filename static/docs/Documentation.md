# Systems

## Install bower

Need `bower` to install polyfills:

```
sudo apt-get install nodejs
sudo apt-get install npm
sudo npm install -g bower
sudo ln -s /usr/bin/nodejs /usr/bin/node
bower install --save webcomponents/webcomponentsjs
```

* [http://stackoverflow.com/questions/21491996/installing-bower-on-ubuntu#22791301](http://stackoverflow.com/questions/21491996/installing-bower-on-ubuntu#22791301)
